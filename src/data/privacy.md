# プライバシーポリシー

quizeum（以下、「本サービス」といいます）は、ユーザーの個人情報の取扱いに細心の注意を払い、以下のプライバシーポリシー（以下、「本ポリシー」といいます）を定めて適切に保護します。

## 1. 収集する情報
本サービスは、以下の情報を取得および収集することがあります。
- アカウント登録時に提供されるメールアドレスおよびパスワード
- プロフィール情報（ユーザー名、プロフィール画像など）
- クイズのプレイ履歴、得点、解答時間などのアクティビティデータ
- お問い合わせフォーム等から送信された内容や連絡先情報
- アクセスログ、クッキー（Cookie）、およびPostHog等を通じたプロダクト分析データ

## 2. 収集した情報の利用目的
収集した個人情報は、以下の目的で利用されます。
- 本サービスの提供、運用、および維持
- ユーザー認証およびセキュリティの確保
- ユーザーからのお問い合わせやサポートへの対応
- 新機能の開発、サービス改善のための分析および統計データの作成
- 有料プラン（Stripe経由での課金処理）の提供と照合

## 3. 第三者への情報提供
本サービスは、法令に基づく場合、または以下に定める場合を除き、個人情報をユーザーの同意なしに第三者に開示または提供しません。
- Firebase (認証・ホスティング・データベースプロバイダ) などのインフラおよびサービス提供に必要な外部委託先
- Stripe (決済プロバイダ) に決済処理を委託する場合
- PostHog (アクセス・分析プロバイダ) に製品分析を委託する場合

## 4. セキュリティ
個人情報の紛失、破壊、改ざん、および漏洩を防ぐため、業界水準のセキュリティ技術を用いて適切な安全対策を講じます。

## 5. 個人情報の開示・訂正・削除
ユーザーは、自身のアカウントページから登録情報の確認、変更、およびアカウントの削除（退会）を行うことができます。また、お問い合わせを通じて追加の開示等の請求を行うことも可能です。

## 6. 本ポリシーの変更
本サービスは、法改定やサービス運用変更に伴い、本ポリシーを随時更新することがあります。最新の本ポリシーは本ページで常時公開されます。
